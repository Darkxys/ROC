Pour ouvrir le jeu, ouvrez simplement le fichier ROC.exe et vous allez être sur la page de connexion.

Sur la page de connexion vous pouvez créer un compte ou vous connecter.

Pour bouger dans le jeu, voici les touches : 
	W : Avancer vers le haut
	A : Avancer vers la gauche
	S : Avancer vers le bas
	D : Avancer vers la droite

Vous pouvez aussi attaquer les taupes sanginaires avec des tirs de flèche. 
Pour ce faire, faites un clique-droit pour lancer une flèche.

Appuyer sur le boutton Q ou E pour augmenter/diminuer la vie.
Appuyer sur le boutton X ou Z pour augmenter/diminuer l'experience.

Vous pouvez aussi changer la base de donnée en utilisant le fichier Config.txt généré lors de la première 
execution du jeu.
